# Homebrew repository for `bcvk`
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fbootc-dev%2Fhomebrew-bcvk.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fbootc-dev%2Fhomebrew-bcvk?ref=badge_shield)


This is packaging for [`bcvk`](https://github.com/bootc-dev/bcvk) on [Homebrew](https://brew.sh) for Linux as this is the only supported platform by that project currently

## How do I get these

```bash
brew install tulilirockz/bcvk/bcvk
```

You can also download the latest dev version by running this instead:

```bash
brew install --head tulilirockz/bcvk/bcvk
```
